리액트란?
- 프론트 웹 개발 프레임워크 
- 자바스크립트와 html을 혼합한 jsx 언어를 사용
- 싱글 페이지, 모바일 앱 개발 시 사용 가능
- 가장 대중성있고 인기있는 라이브러리

컴포넌트란?
- 여러개의 프로그램 함수를 모아 하나의 특정한 기능을 수행할 수 있도록 구성한 기능적 단위

장점 
1. UI 구축을 쉽게 만들어줌
2. 재사용이 용이
3. 유지/보수에 유리

리액트에서 변경한 정보가 화면에 보이게 하려면 state를 변경해서 update 상태로 들어와야함
state를 변경하기 위해 useState를 이용하는게 좋음
state는 객체 저장 방식이기 때문에 직접 객체에 접근해 정보를 수정할 수 없음
따라서 useState 함수를 이용해 새로운 객체를 만들어 할당함

src > component폴더 > jsx파일

App.jsx에서 저장했던 배열을 TodoBoard.jsx로 가져오는 방법? props
배열 안에 값 하나하나를 TodoItem.jsx로 보내주는 방법? props와 map

* map : 반복되는 컴포넌트를 렌더링하기 위해 자바스크립트 배열의 내장함수로
        배열 값 하나하나를 돌며 실행문을 실행문을 실행
* props : 어떤 값을 컴포넌트에 전달해야할 때 사용
